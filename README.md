# 🏨 TripAdda - Luxury Hotel Booking Platform

TripAdda is a premium, full-stack hotel booking platform designed for discerning travelers. It features a bespoke "Midnight Luxury" dark-mode UI with sleek glassmorphism effects, offering a seamless experience for both guests booking luxury stays and hotel owners managing their properties.

## ✨ Features

### 🤵 For Guests (User Facing)
* **Immersive Discovery:** Browse luxury rooms and curated experiences with high-resolution image grids.
* **Smart Booking:** Dynamic check-in/check-out and guest selection tools.
* **Premium UI/UX:** A custom "Midnight Luxury" dark theme featuring frosted glass effects, smooth hover transitions, and cyan/orange neon accents.
* **Secure Authentication:** Integrated with Clerk for seamless, secure user sign-ins and profile management.

### 💼 For Hotel Owners (Dashboard)
* **Revenue & Analytics:** Real-time tracking of total bookings and generated revenue.
* **Room Management:** A sleek interface to add new rooms with image uploads, amenities toggles, and pricing.
* **Booking Tracking:** A clean, glass-morphism table view to monitor pending and completed bookings.
* **Listing Controls:** Easily toggle room availability on and off with custom dark-mode UI switches.

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router DOM
* **Styling:** Tailwind CSS (Custom Dark Theme & Glassmorphism)
* **Authentication:** Clerk (@clerk/clerk-react)
* **Icons & Assets:** Custom SVG assets and high-res Unsplash imagery
